Study Overview

Atristain et al. (2024) investigated whether decommissioning a large dam (42 m tall, Enobieta Dam, northern Spain) restored downstream macroinvertebrate communities. They used a multiple Before-After/Control-Impact (mBACI) design — the ecological equivalent of a difference-in-differences (DiD) framework.

The causal question: Did the reservoir drawdown cause invertebrate communities at impact sites to recover to levels comparable to undammed control sites?

The treatment: Reservoir drawdown (initiated December 2018, completed November 2019).

Primary finding we’ll replicate: The effect of drawdown on taxa richness (S), measured by the period × reach interaction in a linear mixed-effects model.

![](drawdown-diagram.jpg)

NoteData access

The data for this analysis come from the Dryad Digital Repository (Atristain et al., 2024). Download Atristainetal_JAE_Data.xlsx and place it in a data/ folder in your project directory.

The mBACI Design as Difference-in-Differences

The mBACI design maps directly onto the DiD framework:

DiD Term mBACI Equivalent Description
Treatment group Impact sites (I1–I4) Sites downstream of the dam
Control group Control sites (C1–C4) Undammed tributaries & upstream site
Pre-treatment Before period (B) November 2017
Post-treatment Drawdown (D) & After (A) October 2019 & November 2020
Treatment effect Period × Reach interaction DiD estimator

The key assumption — parallel trends — requires that control and impact sites would have followed the same trajectory in the absence of the drawdown. The use of multiple control sites in different tributaries strengthens this assumption.

Step 3: Fit the mBACI Model

The paper uses a linear mixed-effects (LME) model with:

  • Fixed effects: period, reach_type, and their interaction (period:reach_type)
  • Random effect: site_code (to account for repeated measures at the same site across replicates)

The interaction term period:reach_type is the DiD estimator — it captures whether the change over time differed between control and impact sites.

Code 
# fit the mBACI linear mixed-effects model
# this is the exact specification from the paper (Section 2.5)
mod_richness <- lmer(
  richness ~ period * reach_type + (1 | site_code),
  data = invert_baci,
  REML = TRUE
)

Here’s how to read each coefficient:

Coefficient Interpretation
(Intercept) Mean richness for control sites in the Before period
reach_typeI Baseline difference: impact minus control in the Before period (BCI in the paper)
periodDrawdown Change in control sites from Before to Drawdown
periodAfter Change in control sites from Before to After
periodDrawdown:reach_typeI DiD estimator (BD:CI): Did impact sites change differently than controls from Before → Drawdown?
periodAfter:reach_typeI DiD estimator (BA:CI): Did impact sites change differently than controls from Before → After?

The two interaction terms are the key treatment effects. A positive periodAfter:reach_typeI means impact sites gained more richness than controls between the Before and After periods — i.e., they recovered.

Step 5: Test the Overall Interaction (BDA:CI)

The paper also reports a global test of whether the interaction matters at all (Type III test of the period:reach_type interaction). We can get this with anova():

The significant interaction (period:reach_type) tells us that the trajectory of taxa richness over time depended on whether a site was downstream of the dam or not. In other words, the drawdown had a causal effect on invertebrate richness at impact sites.

Step 6: Model Diagnostics

Before trusting the model output, we need to verify that the LME assumptions hold. The key assumptions for a linear mixed-effects model are: (1) residuals are approximately normally distributed and (2) residuals have constant variance (homoscedasticity). If these fail, our p-values and confidence intervals could be unreliable.

Code

We’re looking for: no obvious fan shape or pattern in the residuals vs. fitted plot (homoscedasticity), and points falling roughly along the red line in the QQ plot (normality). Some deviation in the tails is normal with ecological count data — the LME is reasonably robust to mild departures from normality, especially with this sample size.

Step 7: Robustness Checks

A single model on one response variable isn’t enough to make a convincing causal argument. The paper tests the same mBACI specification across multiple community metrics. If the drawdown truly caused ecological recovery, we’d expect the same pattern — a significant period:reach_type interaction — to show up across different measures of community health, not just taxa richness.

We’ll run the same model on three additional response variables: Shannon diversity (H’), total invertebrate density, and the IASPT biomonitoring index (a pollution sensitivity score).

Code 
# Shannon diversity

  data = invert_baci,

# total density (log-transformed, as in the paper)
mod_density <- lmer(
  log(density) ~ period * reach_type + (1 | site_code),
  data = invert_baci,
  REML = TRUE
)

# IASPT biomonitoring index
mod_iaspt <- lmer(
  IASPT ~ period * reach_type + (1 | site_code),
  data = invert_baci,

# extract the period:reach_type interaction row from each anova
extract_interaction <- function(model, response_name) {
  aov <- as.data.frame(anova(model))
  row <- aov["period:reach_type", ]
  tibble(
    Response = response_name,
    `F value` = row[["F value"]],
    `Num DF` = row[["NumDF"]],
    `Den DF` = row[["DenDF"]],
    `p value` = row[["Pr(>F)"]]
  )
}

Step 8: Effect Sizes

To match the paper’s approach, we compute log-ratio effect sizes for each impact site relative to the control mean. This shows how the dam’s effect decreased with distance downstream.

Code 
effect_sizes <- invert_baci |>
  group_by(period, site_code, reach_type) |>
  summarise(mean_richness = mean(richness), .groups = "drop") |>
  # get the control mean for each period
  group_by(period) |>
  mutate(control_mean = mean(mean_richness[reach_type == "C"])) |>
  ungroup() |>
  # compute ln-ratio for impact sites only
  filter(reach_type == "I") |>
  mutate(effect_size = log(mean_richness / control_mean))

Evaluation of Causal Inference

The mBACI/DiD framework makes a causal claim: the drawdown caused the recovery. That claim rests on specific assumptions. Let’s evaluate each one against the evidence.

1. Parallel trends

The assumption: In the absence of drawdown, control and impact sites would have followed the same trajectory over time. This is the foundational assumption of any DiD design — if violated, the “treatment effect” could just be pre-existing divergence.

How well is it met? This is the biggest weakness. There is only one pre-treatment time point (November 2017), so we literally cannot test whether control and impact sites were trending in parallel before the drawdown. The paper partially addresses this by using four independent control sites across different tributaries, which reduces the chance that a single anomalous control drives the result. But it doesn’t eliminate the concern. If, for example, natural recolonization was already underway before the drawdown began, the model would incorrectly attribute that trend to the treatment.

2. No spillover (SUTVA)

The assumption: The drawdown at impact sites didn’t affect the control sites. Each unit’s outcome depends only on its own treatment status.

How well is it met? Reasonably well. Three of the four control sites (C2–C4) are in entirely separate tributaries, so sediment mobilization and water quality changes from the drawdown wouldn’t reach them. C1 is upstream of the dam in the same stream, so it’s also physically isolated from downstream effects. The spatial separation of controls makes SUTVA plausible here.

3. No time-varying confounders

The assumption: Nothing else changed between the Before and After periods that differentially affected impact vs. control sites.

How well is it met? The Artikutza catchment is a protected, forested headwater basin with no other major land use changes during the study period. There were no other restoration actions, no new pollution sources, and no major hydrological changes besides the drawdown itself. This is about as clean a natural experiment setting as you’ll find in ecology. The main risk would be something like an unusually wet or dry year affecting impact and control sites differently, but the paper doesn’t flag any such events.

4. Correct model specification

The assumption: The linear mixed-effects model with site-level random intercepts adequately captures the data structure.

How well is it met? The random intercept for site_code accounts for the fact that five Surber net samples at the same site are not independent — they share site-level characteristics. The residual diagnostics (Step 6) tell us whether the normality and homoscedasticity assumptions hold. One potential concern is that with only 4 sites per group, the random effect variance is estimated from very few clusters, which can lead to anti-conservative p-values. The paper uses REML estimation, which partially corrects for this, but it’s worth noting as a limitation.

Bottom line

The strongest parts of the causal argument are the clean study setting (no confounders) and the spatial separation of controls (SUTVA). The weakest part is the inability to test parallel pre-trends with only one pre-treatment observation. The robustness of the finding across multiple response variables (Step 7) and the spatial gradient in effect sizes (larger effects closer to the dam, as expected) both add indirect support, but they don’t directly address the parallel trends concern.

Summary

We replicated the primary finding from Atristain et al. (2024): the reservoir drawdown caused a significant recovery in invertebrate taxa richness at impact sites. The mBACI design, which is functionally a difference-in-differences estimator with mixed effects, identified this treatment effect by comparing the change over time between impacted and control reaches. Robustness checks across multiple community metrics (Shannon diversity, density, IASPT) all showed the same convergence pattern, strengthening the causal claim. One year after drawdown, impact sites were indistinguishable from controls, suggesting rapid ecological recovery — though the single pre-treatment time point limits our ability to fully rule out pre-existing trends.
